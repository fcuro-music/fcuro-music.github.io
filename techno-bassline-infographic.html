Cメジャーのダイアトニックコードに対応する
テクノ風ベースライン例

2025年3月8日

Cメジャースケール上のダイアトニックコード I, ii, iii, IV, V, vi（それぞれ Cメジャー、Dマイナー、Eマイナー、Fメジャー、Gメジャー、Aマイナー）に対し、テクノらしいベースラインのメロディパターンをいくつか示します。どのパターンも16分音符主体のリズムでシンプルな反復があり、適度なシンコペーション（タイミングのずれ）を含んでいます。パターン自体はキーに依存しない構造ですが、ここでは例としてCメジャーキーで記述します。

I

Cメジャー

オフビートのルート音パターン

最も基本的なパターンで、コードのルート音（C）を主体に据えたものです。4つ打ちキック（各拍の頭）に対し、ベース音を各拍の裏拍（オフビート）で鳴らすことでシンプルながら力強いグルーヴを生み出します。

使用音: ルート音のC。単調さを避けたい場合、時折5度のGを挟んで変化をつけても良いです。

リズム: 1小節（4拍）の各拍の裏で音を鳴らし、拍頭は休符にします。具体的には 各拍の「&」のタイミングでC音を短く鳴らし、それ以外は休符とします。

- C - - | - C - - | - C - - | - C - -
1拍目の裏（1&）、2拍目の裏（2&）、3拍目の裏（3&）、4拍目の裏（4&）でC音が鳴り続け、安定したトニック（主和音）のうねりが生まれます。

ii

Dマイナー

ローリングする16分音符パターン

ii和音（Dマイナー）では、低音域で転がるような16分音符の連打を用いたパターンが効果的です。各拍の頭を休符にし、その後の16分音符で音を埋めることで、キックの隙間を埋める「ローリング」感を出します。

使用音: 主にルート音のD。変化をつけるために5度のAや短3度のFを織り交ぜ、Dマイナーコードの響きを感じさせます。

リズム: 各拍の最初の16分音符を休符にし、その後の3つの16分音符で音を鳴らします。

- D D D | - D A D | - D D D | - D F D
激しく細かい16分音符の連続によって、低音がうねるようなエネルギーを生み出し、Dマイナーコードにマッチしたダークな雰囲気を演出します。

iii

Eマイナー

オクターブ跳躍のパターン

iii和音（Eマイナー）では、オクターブ跳躍を使ったパターンが考えられます。低いルート音と1オクターブ上の同じ音を交互に鳴らすことで、シンプルな音使いでも動きのあるベースラインになります。

使用音: ルート音Eの低音と高音（1オクターブ上のE）。Eマイナーコードの他の構成音はあえて使わず、同じ音程のオクターブ違いだけでミニマルな雰囲気を作ります。

リズム: 各拍の中でオクターブ違いのEを2連打し、残りを休符にするパターンです。

IV

Fメジャー

コードトーンのアルペジオパターン

IV和音（Fメジャー）では、コードトーンを分散和音（アルペジオ）で奏でるパターンが効果的です。明るいメジャーコードの響きを活かし、16分音符の中で音階的な動きを作ります。

使用音: Fメジャーコードの構成音であるF（ルート）、A（長3度）、C（5度）。これらを順番に並べてコードの和音感を表現します。

リズム: 1拍の中で4つの16分音符を使い、コードトーンを上昇下降させるパターンです。

F A C A | F A C A | F A C A | F A C A
このパターンによりFメジャーコードの明るい響きが際立ち、疾走感とメロディアスさを兼ね備えたテクノベースラインが構築できます。

V

Gメジャー

ルートと5度の交互パターン

V和音（Gメジャー）には、ルート音と5度音を交互に繰り返すシンプルなパターンがマッチします。ルート（G）と5度（D）の組み合わせは安定した響きを持ち、いわゆる「パワーコード」的な強さが感じられます。

使用音: ルート音Gと5度音Dのみを使用します。Gメジャーコードの長3度(B)はあえて使わず、安定感のある完全5度のインターバルに絞ることで力強い響きを維持します。

リズム: 8分音符の刻みに沿ってGとDを交互に配置します。

vi

Aマイナー

シンコペーションを効かせたリフパターン

vi和音（Aマイナー）では、シンコペーション（リズムの食い違い）を活かした短いリフ型のパターンが有効です。休符と音符の配置に変化をつけ、単発のフレーズを繰り返すことで独特のノリを生みます。

使用音: Aマイナーコードの構成音であるA（ルート）、C（短3度）、E（5度）を組み合わせます。ルートの反復で土台を作りつつ、短3度や5度を差し込みます。

リズム: 2拍単位のシンコペートしたフレーズを1小節に2回繰り返すイメージです。

- A C - | - E - - | - A C - | - E - -
1小節内での音の並びは「… 1&:A、1a:C、2&:E、3&:A、3a:C、4&:E …」となります。オフビートに細かく音を置くことで独特のノリを生み、Aマイナーの暗く深みのある雰囲気を持ちながらもリズミカルなベースラインになります。
